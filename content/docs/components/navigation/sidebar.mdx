## Usage

```tsx
import * as Sidebar from "@/components/ui/sidebar"
```

Read exported parts in `src/components/ui/sidebar.tsx` and compose the primitive according to the Ark UI pattern for this component.

The API surface and composition (provider, inset, groups, menu, rail, etc.) follow the same mental model as [shadcn/ui’s Sidebar](https://ui.shadcn.com/docs/components/radix/sidebar). For full-page layouts and copy-paste blocks, see the [shadcn sidebar blocks](https://ui.shadcn.com/blocks/sidebar) (sidebar-01 … sidebar-16).
